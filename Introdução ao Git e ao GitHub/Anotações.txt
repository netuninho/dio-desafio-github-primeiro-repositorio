O Git Bash é um terminal extendido para otimizar o uso do Git.

Quando lidamos com terminal sempre colocamos o nome do programa na frente do comando.

Comandos: 
git init: inicia o repositório 
git add: inicia o versionamento
git commit: cria o commit
git status: informa os status da pasta
git clone: cria uma cópia de um repositório
git remote -v: lista os conteúdos do repositório

Atualizando um Projeto no GitHub
git add .
git commit -m "..."
git pull origin main/master
git push origin main/master
